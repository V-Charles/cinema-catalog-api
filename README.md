# 🎬 Sistema de Gerenciamento de Cinema (Spring Boot)

Sistema web desenvolvido para gerenciamento de catálogo de filmes e análises, integrando Backend (Java) e Frontend (Thymeleaf/jQuery).

## 🚀 Tecnologias Utilizadas
- **Java 21** & **Spring Boot 3**
- **Spring Data JPA** & **MySQL**
- **Spring Web (MVC & REST API)**
- **Thymeleaf**, **HTML5**, **CSS3**
- **JavaScript (jQuery & AJAX)**
- **Maven**

## ✨ Funcionalidades
- **CRUD Completo:** Cadastro, Leitura, Edição e Remoção de filmes.
- **API RESTful:** Endpoints consumidos via AJAX para manipulação de dados sem recarregar a página.
- **Sistema de Avaliação:** Relacionamento entre Entidades (Filme <-> Análises).
- **Dark Mode:** Alternância de tema com persistência via Cookies.
- **Arquitetura MVC:** Separação clara de responsabilidades.

## 🛠️ Como executar
1. Clone o repositório.
2. Configure o banco de dados MySQL no `application.properties`.
3. Execute o script `script_banco.sql` (incluso na raiz).
4. Rode a aplicação via IDE ou Maven.
